# APK de Comi2

Tras `npm run cap:apk:debug` (desde `app/`), la APK de prueba se copia aquí:

**`comi2-debug.apk`**

Copia también generada por Gradle en:

`app/android/app/build/outputs/apk/debug/app-debug.apk`

(Esa carpeta está en `.gitignore` y a veces el explorador no la muestra si buscas solo en la raíz del repo.)

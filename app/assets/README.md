# Fuentes para iconos Android

| Archivo | Origen |
|---------|--------|
| `icon.svg` | Copia de `public/favicon.svg` (mismo icono que la PWA) |

Regenerar mipmaps y splash en `android/`:

```bash
npm run cap:icons
```

Fondo del icono adaptativo: `#f7f6f3` (igual que `background_color` del manifest).
